The what

  • Artificial intelligence (AI):
    Umbrella term for computer systems that make smart decisions
  • Machine learning (ML):
    Collection of algorithms that learn to recognise patterns in data
  • Deep learning:
    ML that recognises complex patterns, using neural networks

The big AI tools of today are driven by advancements in

  • Deep learning ‚Äì bigger and better models
  • Computing ‚Äì bigger and better processors

The what

Some terminology:

  • Model:
    A mathematical function (or algorithm) that takes data in and gives predictions out
  • Parameters:
    The internal variables of the model
  • Label:
    The correct answer that the model should learn to predict
  • Supervised learning:
    Learning the relation between data and labels
  • Unsupervised learning:
    Learning patterns without explicit labels

We also need to know some statistics, but let‚Äôs deal with that later.

Training

Optimal choice of model parameters can usually not be found analytically

\(\rightarrow\) need to iteratively search for it, which we call training the model.

Deep learning models are essentially composite, differentiable functions, meaning we can use gradient descent.

Good: DL libraries do the differentiation for us!

Bad: It‚Äôs computationally expensive

Good: Modern hardware (GPUs) are very efficient at this (also, it could have been worse)

Evaluating models

Evaluating a model should be done on an independent data set
(we want to know how well it performs on new, unseen data)

Typically we set aside a part of the data, and use this only for final evaluation.

# "X" are the data, "y" are the targets.
